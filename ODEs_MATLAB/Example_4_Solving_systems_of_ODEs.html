Solving a system of ODEs

The well known SIRD model is a system of 4 Ordinary Differential Equations (ODEs) that model the dynamics of an epidemic, e.g., the flu.

The SIRD function is defined below.
function dydt = SIRD_model(~, y, beta, gamma, mu)
% This function defines the SIRD system of ODEs
% Unpack y array
S = y(1);
I = y(2);
N = sum(y);
% Define SIRD system
dydt = zeros(4, 1);
dydt(1) = -beta * I * S / N;
dydt(2) = beta * I * S / N - gamma * I - mu * I;
dydt(3) = gamma * I;
dydt(4) = mu * I;
end
