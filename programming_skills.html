Programming Skills

Teaching Materials

The materials used to support the delivery of the unit Linear Algebra and Programming Skills at the Manchester Metopolitan University are contained on this page. The two programming languages taught on the unit are MATLAB and Python which are taught using MATLAB Live scripts and Jupyter Notebooks respectively which can be downloaded using the link below.

Link to download of zipped archive

Solutions (MATLAB)

The MATLAB Live scripts distributed to students are incomplete where code cells are left empty for students to complete themselves. The solutions to the exericses can be found using the links below.

  1. MATLAB Basics
    Introduction to MATLAB Live scripts, arithmetic operations, variables, basic programs and formatted output
  2. Symbolic Mathematics
    Symbolic mathematics, symbolic variables, algebra, calculus, solving equations
  3. Arrays
    Defining arrays, array indexing and array operations
  4. If statements
    Logical conditions, Boolean variables and if statements
  5. Loops
    For loops, while loops, loop control and next loops
  6. Functions
    Defining functions, calling functions and inline functions
  7. Plotting
    Line plots, formatting plots, scatter plots, surface plots, 3D plots, contour plots, plotting polygons, image plots
